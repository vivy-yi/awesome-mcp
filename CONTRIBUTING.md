# Contributing to Awesome MCP

Thank you for your interest in contributing to Awesome MCP!

## Quick Start

1. Find a missing MCP project
2. Check [inclusion criteria](#inclusion-criteria)
3. Add to appropriate category
4. Submit a PR

## Inclusion Criteria

**Must have:**
- [ ] GitHub repository with active development
- [ ] Clear README or documentation
- [ ] Relevance to MCP ecosystem
- [ ] Not a fork of already listed project

**Nice to have:**
- [ ] Official support or endorsement
- [ ] Active maintenance (commits within 6 months)
- [ ] Clear documentation

## Format

### For tables (3+ items with consistent attributes)

```markdown
| Project | Stars | Description |
|---------|-------|-------------|
| [user/repo](url) | ![Stars](badge) | Description |
```

### For lists

```markdown
- [user/repo](url) - Brief description
```

### Stars Badge Format

```markdown
![Stars](https://img.shields.io/github/stars/user/repo)
```

## Categories

- **Official** - MCP specification, official SDKs, reference implementations
- **Core Servers** - Major platform servers (AWS, Microsoft, GitHub, etc.)
- **Cloud Services** - Cloud platform integrations
- **Database** - Database-related MCP servers
- **Browser & Automation** - Browser control and automation
- **Search & Scraping** - Web search and content extraction
- **Development Tools** - Developer tooling integrations
- **Communication** - Messaging platforms
- **Productivity** - Productivity tools
- **Clients** - MCP client applications
- **Frameworks & SDKs** - Development frameworks
- **Resources** - Documentation and learning materials

## Review Process

- PRs are reviewed within 1 week
- Please ensure links are working and stars badges are correct
- Multiple small PRs preferred over one large PR
